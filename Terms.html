ChromeOS Terms

----- ChromeOS -----

ChromeOS is an operating system(OS) designed and developed by Google It is derived from the open-source ChromiumOS operating system, which itself is derived from Gentoo Linux

----- Chromebooks -----

Chromebooks are a laptops running ChromeOS. Chromebooks are designed to be used primarily while connected to the Internet, with most applications and data residing in the cloud. (i.e the Dell Chromeook 3110 only has 32 gb oqf storage, but you can use google drive for more storage)

----- ChromeOS Bootloader -----

ChromeOS Bootloader is a Bootloader that is used to boot chromeOS on startup, it is very locked down and hard to modify, it is what makes it hard to install other OS on Chromebooks.

----- ChromeOS Recovery Screen-----
ChromeOS Recovery Screen is a screen that appears when a Chromebook is unable to boot into ChromeOS. It has opations to enable Dev Mode, Do test's, and Recover ChromeOS. You can manually access it by pressing Esc + Refresh + Power button.

Hardware Terms

----- CPU -----

Centralized Processing Unit (CPU) is the primary processor in a given computer. Its electronic circuitry executes instructions of a computer program, such as arithmetic, logic, controlling, and input/output

----- GPU -----

Graphics Processing Unit (GPU) is a specialized electronic circuit designed for digital image processing and to accelerate computer graphics, being present either as a component on a discrete graphics card or embedded on motherboards (Most ChromeBooks have this option), mobile phones, personal computers, workstations, and game consoles.

----- RAM -----

Random Access Memory (RAM) is a form of computer data storage that stores data and machine code currently being used. A random-access memory device allows data items to be read or written in almost the same amount of time irrespective of the physical location of data inside the memory. (Most Chromebooks have 4gb of RAM, but some have 8gb)

File Types

----- .crx -----

.CRX is a file extension for a Google Chrome extension (Chrome extensions are small software programs that customize the browsing experience). Thoe Many School Districts Block .CRX files from being installed on school Chromebooks.

----- .zip -----

.ZIP is a file format that supports lossless data compression. A ZIP file may contain one or more files or directories that may have been compressed.

----- .py -----

.PY is a file extension for Python files. Python is a high level prpgarming language that is used for most things(i.e Youtube, Severs, and Ai).

----- .js -----

.JS is a file extension for JavaScript files. JavaScript is a high-level compiled language that is used in Websites, Web Applications, and Servers.

----- .html/css -----

.HTML/CSS are file extensions for HTML and CSS files. They are languages used to make websites. HTML is used for the structure of the website, and CSS is used for the style of the website. (remember HTML is the Bones, CSS is the skin and clothes, and javascrpit is the Soul)

Programming Languages

----- Python -----

Python is a high level general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically type-checked and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming.

----- JavaScript -----

JavaScript is a programming language and core technology of the web platform, alongside HTML and CSS. Ninety-nine percent of websites on the World Wide Web use JavaScript on the client side for webpage behavior.

----- C -----

C is a general-purpose programming language. It was created in the 1970s by Dennis Ritchie and remains widely used and influential. By design, C gives the programmer relatively direct access to the features of the typical CPU architecture, customized for the target instruction set. It has been and continues to be used to implement operating systems (especially the ChromeOS kernel)

Acronyms

----- SPI Chip (In Chromebooks) -----

Serial Peripheral Interface (SPI) is a chip that stores the ChromeOS Bootloader, and other important information. It is a very small chip that is soldered onto the motherboard of the Chromebook. (It's the Thing that Dictates if you can into dev mode, or if you can setup your chromebook without school management)

----- TPM (In Chromebooks) -----

Trusted Platform Module (TPM) is a specialized chip on an endpoint device that stores RSA encryption keys specific to the host system for hardware authentication. (It's the thing that makes sure your Chromebook is secure, and if you can use certain features, like Linux or Android Apps)
